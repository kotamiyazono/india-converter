# 🇮🇳 インド単位変換アプリ

インドの通貨と面積の単位を簡単に変換できるPWA（Progressive Web App）モバイルアプリです。

## 機能

### 💰 通貨モード
- **日本円（¥）** ⇄ **インドルピー（₹）**
- **ラック**（1 ラック = 100,000 ルピー）
- **クロール**（1 クロール = 100 ラック = 10,000,000 ルピー）
- 為替レートはアプリ内の設定画面で変更可能
- 設定した為替レートはLocalStorageに保存され、次回起動時も維持されます

### 📐 面積モード
- エーカー（acre）
- 平方フィート（sq ft）
- ヘクタール（ha）
- 平方メートル（m²）
- 坪

## 使い方

### 起動方法

1. ローカルサーバーを起動します：

```bash
# Python 3を使用する場合
python3 -m http.server 8000

# もしくは
python -m http.server 8000
```

2. ブラウザで `http://localhost:8000` にアクセス

### PWAとしてインストール

- Chrome（PC/Android）: アドレスバーの右側にある「インストール」ボタンをクリック
- Safari（iOS）: 共有ボタン → 「ホーム画面に追加」

### 使用方法

1. **タブ切り替え**: 「💰 通貨」または「📐 面積」タブをクリック
2. **単位変換**: いずれかの入力フィールドに数値を入力すると、自動的に他の単位に変換されます
3. **為替レート設定**: 右上の⚙️ボタンをクリックして為替レートを変更できます

## 技術仕様

- **フロントエンド**: HTML5, CSS3, Vanilla JavaScript
- **PWA機能**: Service Worker, Web App Manifest
- **オフライン対応**: 一度アクセスすれば、オフラインでも使用可能
- **レスポンシブデザイン**: スマートフォンに最適化
- **データ保存**: LocalStorage（為替レート設定）

## 換算レート

### 通貨（デフォルト: 1 JPY = 0.6 INR）
- 1 ラック = 100,000 ルピー
- 1 クロール = 100 ラック = 10,000,000 ルピー

### 面積（平方メートル基準）
- 1 m² = 10.7639 sq ft
- 1 m² = 0.000247105 acre
- 1 m² = 0.0001 hectare
- 1 m² = 0.3025 坪
